Development of a Low-Cost Quadruped Robotic Dog Companion for Visually Impaired Navigation
-------

Objective 
- 
To design a quadruped robot that moves alongisde the user, detects obstacles using sensors, maintains stable locomotion, and helps the user navigate. 
Visually impaired individuals rely on tools like canes or guide dogs for navigation, but these have limitations.
Advanced robotic systems exist but are often expensive and computationally intensive. So, this robotic dog will be a low-cost robotic assistant without requiring fully autonomous navigation. 

Hardware
- 
-Servo Driven legs (12 servos)

-PCA9685 Servo Driver

-Rasbperry Pi

-Buck Converter
  
-IMU sensor (tilt/stability)  

-Camera/Ultrasonic Sensor 

Software 
-
- Python

- Inverse Kinematics for walking

- OpenCV for object detection

- Integrate GPS/mapping in future
